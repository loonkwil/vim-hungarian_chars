# hungarian\_chars.vim

Useful functions for special Hungarian characters.
Right now, there is two function:

 1. `FixSpecialHungarianChars` will fix charaters like õ, û
 2. `TransliterateSpecialHungarianChars` will replace all special hungarian
    letter

## Installation

If you use [pathogen.vim](https://github.com/tpope/vim-pathogen):
```
cd ~/.vim/bundle
git clone git://github.com/loonkwil/vim-hungarian_chars.git
```

If you use [Vundle](https://github.com/gmarik/vundle) add the line
`Bundle 'loonkwil/vim-hungarian_chars'` to your .vimrc
